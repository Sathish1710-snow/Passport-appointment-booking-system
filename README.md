# Project 6 - Passport-appointment-booking-system
Automating passport appointment booking to reduce manual effort and improve user experience.

Project Purpose:

The objective of this project is to digitize and automate the passport appointment process, reducing manual effort, improving efficiency, and enhancing user experience through structured workflows and automated communication.

Key Features:
- Designed a Service Catalog Item for users to request passport appointments
- Implemented date & time slot selection for booking appointments
- Automated request handling using Flow Designer
- Integrated approval workflow for appointment validation
- Configured email notifications for booking confirmation, approval, and rejection
- Ensured a smooth and user-friendly experience through automation

Technology Used:
- ServiceNow Service Catalog
- Flow Designer
- Email Notifications

Key Learnings:
- Hands-on experience with Flow Designer automation
- Designing real-time business use cases using Service Catalog
- Implementing event-driven notifications
- Building scalable and user-friendly solutions in ServiceNow
